<Section type="about">
Use this provider to authenticate with Yahoo.

```ts {5-8}
import { YahooProvider } from "@openauthjs/openauth/provider/yahoo"

export default issuer({
  providers: {
    yahoo: YahooProvider({
      clientID: "1234567890",
      clientSecret: "0987654321"
    })
  }
})
```
</Section>
