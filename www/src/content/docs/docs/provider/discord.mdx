<Section type="about">
Use this provider to authenticate with Discord.

```ts {5-8}
import { DiscordProvider } from "@openauthjs/openauth/provider/discord"

export default issuer({
  providers: {
    discord: DiscordProvider({
      clientId: "1234567890",
      clientSecret: "0987654321"
    })
  }
})
```
</Section>
