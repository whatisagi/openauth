<Section type="about">
Use this provider to authenticate with Github.

```ts {5-8}
import { GithubProvider } from "@openauthjs/openauth/provider/github"

export default issuer({
  providers: {
    github: GithubProvider({
      clientID: "1234567890",
      clientSecret: "0987654321"
    })
  }
})
```
</Section>
