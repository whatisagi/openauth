<Section type="about">
Use this provider to authenticate with X.com.

```ts {5-8}
import { XProvider } from "@openauthjs/openauth/provider/x"

export default issuer({
  providers: {
    x: XProvider({
      clientID: "1234567890",
      clientSecret: "0987654321"
    })
  }
})
```
</Section>
